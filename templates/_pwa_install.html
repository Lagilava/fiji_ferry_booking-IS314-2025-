{% comment %} Floating "Install App" prompt for the PWA (manifest.webmanifest + sw.js already wired in base.html). Mobile-only by CSS gate — desktop browsers can still install via their own address-bar icon. Handles three cases: 1. Android/Chrome/Edge — captures `beforeinstallprompt`, button triggers the native install dialog. 2. iOS Safari — no `beforeinstallprompt` support at all, so we detect iOS + "not already installed" and show a small instruction sheet instead. 3. Already installed (standalone display mode) — button never shows. Dismissal is remembered for 14 days so it doesn't nag returning visitors. {% endcomment %}
